# stockfishArm

This robotic arm physically plays chess with you on a custom chess set. To play, simply set up the pieces on the board and play the first move.

I’ve been playing chess since 7th grade. While playing against the Stockfish engine has been a good way for me to improve by myself, it’s obviously a lot more fun to play against someone on a real-life board. This idea combines the convenience of Stockfish and the experience of a physical chess set.
